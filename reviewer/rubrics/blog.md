# Blog + X rubric — the "It's Already ___" family

Applies to: It's Already Written (TTRPG), It's Already Priced (crypto security + markets),
It's Already When (cyber). Each has a Jekyll blog on GitHub Pages + a paired X account
that tweets on publish. Voice is knowing, security-led, "you already know how this ends."

Score each criterion 1–5 (5 = excellent) with a one-line justification and a concrete fix.

## Blog article (most recent post)
1. **Headline** — Sharp, specific, on-theme with the blog's three-word conceit; earns the click.
2. **Opening** — First paragraph delivers the thesis fast; no throat-clearing.
3. **Structure & depth** — Skimmable (headers/lists) yet substantive; real insight, not filler.
4. **Lane fit** — On-topic for THIS blog's beat (TTRPG rules / crypto-security / cyber). Flag drift.
5. **Internal linking & CTA** — Links to related posts/issues/RTFM; clear next read.

## X promotion (most recent tweet/thread)
6. **Hook tweet** — Stops the scroll; not a bare "new post" link.
7. **Thread/value** — If a thread, each tweet earns its place; standalone value before the link.
8. **Voice consistency** — Matches the blog's knowing, dry, security-led tone.

Also surface: the single highest-leverage improvement, any lane drift, and whether the X post
is doing real promotion vs. an auto-generated link dump.
